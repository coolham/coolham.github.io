发表于 2023-03-12

ChatGPT 可以生成的 11 种强大的输出格式

1. Graph

Example Prompt:

Create a line graph showing the Google search trends for a particular AI over the past year.

2. Checklist

Example Prompt:

Create a checklist for preparing for a job interview.

3. Scorecard

Example Prompt:

Can you generate output in a scorecard format on how apple stocks performed against tesla in past 5 years.

4. JSON

Example Prompt:

Can you provide details of the last 2 FIFA world cup winners in JSON format?

I want the following to include a date, opponent name, winner name, score, and stadium name.

5. Chart

Example Prompt:

Johnny made $50k sale in Jan, $40K in Feb and $30k in March.

Shaun made $60k sale in Jan, $20K in Feb and $75k in March.

Can you put this data in a bar chart?

6. Table

Example Prompt:

Can you provide details of the last 2 FIFA world cup winners in JSON format?

I want the following to include a date, opponent name, winner name, score, and stadium name.

7. Code

Example Prompt:

Write { Code you want to generate } in { programming language }

8. CSV

Example Prompt:

Can you provide details of the last 2 ICC cricket world cup winners in CSV format?

I want the following to be included in it a date, opponent name, winner name, score, stadium name.

9. XML/ YAML

Example Prompt:

Give me data on the USA population for the years 2012 and 2013. I want the total count with males and females breakdown in XML and YAML format.

10. SVG

Example Prompt:

Create SVG of a smiling face.

11. Diagrams

Example Prompt:

Can you provide me with a diagram of the solar system?
